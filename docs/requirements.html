Requirements to Run Binky

PERL - http://www.perl.org/
This is the language Binky is written in, so you gotta have it. Binky also requires some PERL Modules to be installed on the system.
A web server - This can be a separate machine
Binky is configured only through a web-based interface. Please note that the web server must be able to run PERL files, but it only needs a few modules. The web server must be able to run PERL scripts as CGI files. [More Info ...]
A database - This can be yet another machine
Binky's configuration, logs, and everything else is stored in a database whenever possible. I use MySQL, but I bet that Binky can be ran with nearly any other sort of database.
